# test

A simple test repository demonstrating basic testing functionality with Python.

## Contents

- `calculator.py` - Simple calculator module with basic arithmetic operations
- `test_calculator.py` - Test suite for the calculator module

## Running Tests

To run the tests, use Python's built-in unittest module:

```bash
python -m unittest test_calculator.py -v
```

## Features

The calculator module supports:
- Addition
- Subtraction
- Multiplication
- Division (with zero-division error handling)
